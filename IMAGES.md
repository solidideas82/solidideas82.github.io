# Instructions for Adding Images

## Adding h-nebula-render.jpg
This document outlines the steps to add the h-nebula-render.jpg image file to the repository.

### Recommended Image Specifications:
- **Resolution:** 1200 x 800 pixels
- **Format:** JPEG

### Steps to Upload via GitHub Interface:
1. Navigate to the `solidideas82/solidideasaero-web` repository on GitHub.
2. Click on the `Add file` button, then select `Upload files`.
3. Drag and drop the h-nebula-render.jpg file or click `Choose your files` to select it from your device.
4. After the upload completes, scroll down to the `Commit changes` section.
5. Enter a commit message (e.g., "Add h-nebula-render.jpg image").
6. Select the option to commit directly to the `main` branch (or create a new branch for the changes).
7. Click on `Commit changes`.

### Steps to Upload via Git Command Line:
1. Open your terminal or command line interface.
2. Navigate to your local repository folder:
   ```bash
   cd path/to/your/local/repo
   ```
3. Copy the h-nebula-render.jpg image to your local repository folder.
4. Add the image to your git staging area:
   ```bash
   git add h-nebula-render.jpg
   ```
5. Commit your changes with a message:
   ```bash
   git commit -m "Add h-nebula-render.jpg image"
   ```
6. Push your changes to the remote repository:
   ```bash
   git push origin main
   ```

By following these instructions, you will successfully add the h-nebula-render.jpg image to the repository.
