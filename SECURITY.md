# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it to us immediately.

### How to Report

**Email:** Andres@solidideasaero.com  
**Phone:** 770-901-1694

### What to Include

Please provide the following information in your report:
- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Any suggested fixes (if available)

### Response Timeline

- You will receive an acknowledgment within **48 hours**
- We will investigate and provide an initial assessment within **5 business days**
- We will keep you updated on the progress of addressing the vulnerability
- Once resolved, we will credit you for the discovery (unless you prefer to remain anonymous)

## Scope

This security policy applies to:
- The public website code in this repository
- Any deployed instances of this website

**Note:** This repository contains only public-facing website code. All proprietary technical data, client information, and trade secrets are protected under separate secure systems and NDA protocols.

## Security Best Practices

We follow these security practices:
- Regular dependency updates
- Code review for all changes
- HTTPS/SSL for all deployments
- No sensitive data in repository
